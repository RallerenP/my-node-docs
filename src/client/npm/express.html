Express



Express is an NPM package used for creating web-servers.

Express has many features, but this page will describe the core features I've used.

Installation

The easiest way is to just install as a dependency with NPM

Terminal 
    
        npm install express --save


Then then an app.js should be created, which is a common name for express applications.

app.js 
    const express = require('express');
    ...


Usage

First express should be imported, and an app object made.

app.js 

    const express = require('express');
    const app = express();
    ...


While express can be run without one, it doesn't make much sense, so at least a route and a handler should be created

app.js 
    ...
    app.get('/greeting', (req, res) => {
        res.send("Hello World!");
    })
    ...


And finally tell express to listen for requests on a port.

app.js 
    ...
    app.listen(3000, () => {
        console.clear();
        console.log(`Listening on ${3000}...`);
    });


Running the server (see autoreload) and sending a GET request to localhost:3000/greeting yields 

    GET localhost:3000/greeting
    >> Hello World!


For some common features use

app.js 
    ...
    app.get('/', (req, res) => {
        res.sendFile(path);
    })
    ...


To send a file (like an HTML file)

or

app.js 
    ...
    app.use(express.static(path))
    ...


To serve a static directory in the root of the server.
