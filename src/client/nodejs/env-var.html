Environment Variables



NodeJS can access environment variables for the process.

To access a variable in your project,

app.js 

    ...
    const foo = process.env.VARIABLE_NAME;
    ...


This could be used for many purposes, for example setting different ports for the development environment and productions environment.

app.js 
    ...
    const PORT = process.env.PORT || 80;
    //...
    app.listen(PORT, () => {
        console.clear();
        console.log(`Listening on ${PORT}...`);
    });
    ...


cross-env

An easy way to do this, is to install the cross-env NPM package.

Terminal 

    
        npm install cross-env --save-dev


Cross-env is a tool that can set environment variables across different Operating Systems.

cross-env is run either in the terminal or as a script with

Terminal 
    
        cross-env KEY=VALUE [command]
